## About the Project

ZAZpk.com is designed to provide a comprehensive, secure, and scalable online shopping experience for customers across Pakistan.

Core objectives:
- Simple and intuitive navigation
- Fast product discovery and search
- Secure checkout and payment processing
- Reliable order tracking and delivery
- Customer reviews and feedback
- Promotions, discounts, and free-shipping campaigns

The platform supports multiple product categories and is engineered with scalability, security, and maintainability in mind.

---

## Key Features

### Frontend
- Responsive UI (desktop, tablet, mobile)
- Product listing with filters and search
- Persistent shopping cart
- Multi-step checkout flow
- User dashboard (orders, profile, wishlist)
- Product reviews and ratings
- Discount banners and promotions
- Payment integrations (Credit Card, JazzCash, EasyPaisa)

### Backend
- RESTful API architecture
- JWT authentication with refresh tokens
- Role-based authorization (Admin, Customer)
- Product, category, cart, and order management
- Inventory and stock tracking
- Image uploads with Cloudinary
- Email notifications (order, password reset)
- Analytics and reporting endpoints

## Technology Stack

### Frontend
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS, CSS Modules
- Zustand (state management)
- Axios
- React Hook Form + Zod
- Radix UI
- Lucide React icons

### Backend
- NestJS
- TypeScript
- PostgreSQL
- TypeORM
- Passport.js + JWT
- Multer + Cloudinary
- Nodemailer
- Redis (optional)
- Swagger / OpenAPI

### DevOps & Infrastructure
- Docker & Docker Compose
- Git & GitHub
- GitHub Actions (CI/CD)
- Vercel (Frontend)
- AWS / Hetzner (Backend)
- AWS RDS / Managed PostgreSQL
- Cloudinary (media storage)
